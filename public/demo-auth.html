Instruções para Autenticação Demo

Utilize o QR Code abaixo para autenticar o Scanner Demo no app Zupy Scanner. Com ele, é possível testar todas as funcionalidades de demonstração sem afetar dados reais, login ou autenticação adicional. Não é necessário criar conta nem fornecer informações extras. Este QR Code é exclusivo para testes e revisão do app.

QR Code de Autorização do Scanner Demo
Como usar:
  • Abra o app Zupy Scanner no seu dispositivo.
  • Na tela de autenticação, escaneie o QR acima.
  • O app será autenticado como Scanner Demo, pronto para testes e revisões.
  • Você pode repetir este processo em quantos dispositivos desejar.

Dúvidas ou problemas? Entre em contato com o suporte Zupy.
